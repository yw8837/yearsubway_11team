# 외부데이터 출처

분석에 Join한 외부데이터의 출처·기준연도. (인구 데이터 Join 키 = **자치구**, 기상 Join 키 = **연-월**)

| 파일 | 내용 | 출처 | 기준 |
|---|---|---|---|
| `서울특별시_자치구별 연령별 주간 야간 인구_20201231.xlsx` | 자치구별 **주간인구지수**(주간/상주×100) | 공공데이터포털 (통계청 인구총조사) | 2020 |
| `직업유형별+취업인구(구별)_*.csv` | 자치구별 직업군 취업인구 | 서울 열린데이터광장 | 2020 |
| `기온_*.csv` / `강수_*.csv` | 서울 월별 평균기온·강수량 | 기상청 기상자료개방포털 (data.kma.go.kr) | 2015–2021 *(월별 재다운로드 예정)* |

## 활용
- **주간인구지수** → EDA의 방향성(업무지구 vs 베드타운) 가설을 실제 인구로 실증
- **직업 취업인구** → 자치구 성격(업무 밀집도) 보조 근거
- **기상** → 월별 이용량과 계절·날씨 관계 (시계열 보조)

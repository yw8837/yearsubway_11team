# 외부데이터 출처

분석에 Join한 외부데이터. (인구 Join 키 = 자치구, 기상 Join 키 = 연-월)

| 파일 | 내용 | 출처 | 기준 |
|---|---|---|---|
| `서울특별시_자치구별 연령별 주간 야간 인구_20201231.xlsx` | 자치구 **주간인구지수** | 공공데이터포털(통계청 인구총조사) | 2020 |
| `기온_*.csv` / `강수_*.csv` | 서울 **월별** 평균기온·강수량 | 기상청 기상자료개방포털 (자료구분 월·기본) | 2015–2021 |

## 활용
- **주간인구지수** → 방향성(업무지구 vs 베드타운) 실증 (상관 −0.67)
- **기상(시계열)** → 월별 이용 ↔ 기온·강수: 2020 코로나가 지배적, 날씨는 부차(상관≠인과)

## 제외한 데이터
- 직업유형별 취업인구: **대분류만**이라 세부 직군 구분 불가 + 주간인구지수와 메시지 중복(직장인구 절대값은 큰 베드타운이 Top이라 방향성 왜곡) → 미사용
